# Lounas tänään

Sovellus, joka kerää päivittäiset lounaslistat lähialueen ravintoloista ja näyttää ne yhdessä paikassa. Saatavilla sekä internet selaimessa, että mobiilisovelluksena.

---

## Ominaisuudet

- Päivittäiset lounasmenut automaattisesti ravintoloiden verkkosivuilta
- Karttanäkymä ravintoloista sijainteineen
- Listanäkymä etäisyysjärjestyksessä
- Ravintoloiden arvostelu tähdillä ja kommenteilla
- Kirjautuminen Supabase-autentikaatiolla
- Admin-sivu menujen ja ravintoloiden hallintaan
